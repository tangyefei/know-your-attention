# Know Your Attention

> 一个面向「频繁切换工作」场景的轻量级注意力管理工具。

![image.png](https://my-obsidian-images-bucket-2026.oss-cn-shanghai.aliyuncs.com/img/20260519212650435.png)

## 项目简介

在日常工作中，我们常常被各种事项打断：写代码时被 IM 拉去对需求，刚切到会议又被叫去看告警，回到原任务时已经忘了上下文。

**真正消耗精力的，往往不是事项本身，而是在不同事项之间来回切换的认知成本。**

Know Your Attention 试图帮助你解决三个问题：

1. **降低记录与切换成本**
用尽可能少的操作，快速标记「我现在正在处理什么」

2. **看见时间都花在哪了**
自动按「进行中」状态累计每个事项的实际投入时长

3. **让重要上下文不丢失**
支持置顶、备注和右侧进展面板，切回任务时可快速恢复上下文

---

## 目标用户

- 同时挂着 3+ 项工作、经常被打断的开发者 / 设计师 / PM
- 想了解自己一天时间都花到哪儿了的知识工作者
- 喜欢用键盘完成大多数操作的效率型用户

## 功能概览

项目围绕 **事项**、**随手记**、**设置** 三个模块组织。

### 事项管理

- 支持录入、添加子项、编辑
- 编辑父项时可点击 emoji 快速替换
- 状态流转：未开始 → 进行中 → 暂停
- 同一时刻仅允许一个事项处于「进行中」
- 进行中事项高亮显示，子项与父项时间展示为绿色
- 时长自动累计，仅统计「进行中」时段
- 父项时长 = 自身时长 + 子项时长之和
- 支持顶层事项置顶，并独立分组展示
- 顶部开关可一键显示 / 隐藏已完成事项
- 支持数字徽章 `1~9` 快速跳转
- 选中事项后，右侧会出现独立备注面板，并自动持久化

### 随手记

![image.png](https://my-obsidian-images-bucket-2026.oss-cn-shanghai.aliyuncs.com/img/20260519212714484.png)

随手记与事项完全解耦，适合临时备忘、快速记录灵感或待办，独立 Tab、独立列表管理。

### 设置

![image.png](https://my-obsidian-images-bucket-2026.oss-cn-shanghai.aliyuncs.com/img/20260519212728548.png)

设置集中在右上角 `⋯` 菜单中：

- **防窥模式**：通过滑块控制窗口失焦时的整体不透明度，默认 `100%`（不防窥）
- **显示已完成**：与顶部开关同步
- **导出 Markdown / JSON**：
支持「选择导出文件夹」。在 Chrome / Edge 等支持 File System Access API 的浏览器中，会弹出系统级保存对话框；其他浏览器则回退为默认下载
- **导入 JSON**：与现有数据按 `id` 合并去重，已有备注优先保留
- **重置数据**：二次确认，建议先导出备份

## 数据存储

所有数据均存储在 `localStorage` 中，刷新或关闭后不会丢失。

| Key | 内容 |
|---|---|
| `kya.tasks` | 事项与子项（状态、累计时长、置顶时间、展开态） |
| `kya.notes` | 随手记 |
| `kya.taskNotes` | 每个事项的右侧备注 |
| `kya:peek-opacity` | 防窥滑块值 |
| `kya:show-done` | 是否显示已完成 |
| `kya:shortcut-collapsed` | 快捷键浮层收起状态 |

---

## 快捷键

右下角提供常驻快捷键浮层，可点击 `▾` 收起为底部胶囊。
设计原则：**优先使用裸键，并尽量避免与浏览器 / 系统默认快捷键冲突。**

![image.png](https://my-obsidian-images-bucket-2026.oss-cn-shanghai.aliyuncs.com/img/20260519212748057.png)

| 范围 | 快捷键 | 操作 |
|---|---|---|
| 全局 | `N` | 新增事项（选中父项时为子项） |
| 全局 | `Shift + N` | 强制新增父项（顶层） |
| 全局 | `[` / `]` | 切换上一个 / 下一个 Tab |
| 事项 | `↑` / `↓` | 上下移动选中项（循环） |
| 事项 | `←` / `→` | 收起 / 展开子项 |
| 事项 | `Shift + ←` | 跳转到父级（循环） |
| 事项 | `Shift + →` | 跳转到第一个子项 |
| 事项 | `1` ~ `9` | 跳转到第 N 项 |
| 事项 | `Space` | 切换进行中 / 暂停 |
| 事项 | `X` | 标记完成 |
| 事项 | `H` | 显示 / 隐藏已完成事项 |
| 事项 | `Enter` | 进入编辑 |
| 编辑 | `Cmd + Enter` | 提交 / 保存 |
| 编辑 | `Esc` | 取消 / 退出编辑 |

## 技术栈

- React 18 + TypeScript
- Vite 5
- 纯 CSS 实现，无 UI 框架
- 数据全部存储在 `localStorage`
- 零第三方依赖（emoji、状态图标等均内置）

---

## 设计原则

- **低打扰**：尽量减少界面噪音与操作成本
- **高可见**：让任务状态与时间投入一目了然
- **可恢复**：帮助用户快速回到中断前的上下文
- **键盘优先**：让高频操作尽可能脱离鼠标
